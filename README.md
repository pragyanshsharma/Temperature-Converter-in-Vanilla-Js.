# Temperature-Converter-in-Vanilla-Js.
I made a simple temperature converter using pure HTML 5, CSS 3, Vanilla Js.
HTML: Simple tags like div,heading,meta,links,classes,ids.
CSS3: to style, box-shadow,border, flex, linear-gradient, alignment,etc.
Javascript: fetched, +event.target.value to get text and convert to string, switch statement to work on all 3 cases(celsius, kelvin, fahrenheit), used formulas for conversion, toFixed() for rounding to decimal places, breaks to stop.
